OpenWeather API

You can call this service with an appropriate Country (Using ISO 3166 country codes) and City to receive a weather forecast for the city.

Sample Query

The api takes two parameters. Country and City. In addition to this you must include a header named apikey with your API Key to authenticate against the API.

https://openweather.anthonymurphy.net/api?Country=AU&City=Burwood
